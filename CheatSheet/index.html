HTML Tags CheatSheet

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The display of attributes put into the table.
<tfoot> Table Footer The set of rows defining the footer in a table.

List Tags

Tag Name Description
<ul> Unordered List Creates an unordered list that is not numbered.
<ol> Ordered List Creates an numbered list in the order inputted.
<li> List Item Describes items in lists.

Misc. Tags

Tag Name Description
<br> Line Break Self-closing tag that simulates pressing enter (spaces an element.)
<em> Emphasis Puts lettering in italics within HTML. Can be used differently in CSS.
